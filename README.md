# MMO Tycoon 98

Jogo estilo Windows 98 feito em HTML/CSS/JS puro, num único arquivo.

## Como subir no GitHub Pages

1. Crie um repositório novo no GitHub (ex: `mmo-tycoon-98`).
2. Suba o arquivo `index.html` pra raiz do repositório (é importante que o nome seja `index.html` pro GitHub Pages reconhecer automaticamente).
3. Vá em **Settings > Pages** no repositório.
4. Em "Source", selecione a branch `main` (ou `master`) e a pasta `/ (root)`.
5. Salva e espera uns minutinhos — o link vai aparecer ali mesmo em Settings > Pages, algo tipo:
   `https://seu-usuario.github.io/mmo-tycoon-98/`

Pronto, é só isso — como é um único HTML autocontido (sem dependências externas de build), não precisa de nenhum passo de compilação.
